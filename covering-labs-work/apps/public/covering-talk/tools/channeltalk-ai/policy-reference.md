# 커버링 서비스 정책 종합 레퍼런스

> 54개 청크 요약(5,332 Q&A 쌍)에서 추출한 모든 구체적 정책, 규칙, 숫자, 절차, 예외사항.
> AI 검증용 기준 문서.

---

## 1. 서비스 개요 (커버링이란)

### 서비스 정의
- **커버링**: 생활폐기물 야간수거 서비스
- 고객이 앱으로 수거를 신청하면, 야간(밤 10시~오전 6시)에 기사가 방문하여 봉투를 수거하고, 무게 측정 후 요금을 청구하는 구조
- 선 수거, 후 결제 방식

### 서비스 지역
- 현재 수도권 + 천안 + 세종 + 청주 + 대전 지역 운영
- 서울: 매일 수거 가능
- 인천 서구: 신현원창동 제외 지역 매일 수거
- 세종: 조치원읍, 한솔동, 도담동 등 (화/금/일)
- 청주시: 상당구, 서원구, 흥덕구, 청원구 — 오송읍 포함
- 대전: 서구, 유성구, 동구, 중구, 대덕구 — 전 구 서비스
- **미지원 지역 예시**: 대구, 경남 밀양, 경기 양평, 부산 동래구, 충남 아산 음봉면, 인천 신현원창동
- 경기 고양시 덕양구: 효자동, 고양동, 관산동 서비스 불가
- 지역별 수거 가능 요일이 상이함 (앱 > 내정보 > 서비스 지역에서 확인)
- 서비스 미지원 지역 회원은 탈퇴 없이 계정 유지 시, 지역 오픈 때 신규 쿠폰 재발급

### 상담 채널
- **유선 상담 불가** -- 채팅 상담만 가능 (채널톡)
- 상담 운영시간: 오전 10시 ~ 오후 10시 30분
- 종료된 채팅은 누락 위험이 있으므로 새로운 채팅으로 재문의 권장
- 문의량 급증 시 답변 지연 안내

## 2. 수거 규칙 (시간, 봉투, 무게, 배출 방법)

### 수거 시간
- **수거 시간대**: 오후 10시(22:00) ~ 익일 오전 6시(06:00)
- **당일 수거 신청 마감**: 오후 10시 이전 신청 필수
- 오후 10시 이후 신청 시: 익일 수거
- 수거일은 신청일의 해당 날짜 (예: 목요일 신청 시 목 오후 10시~금 오전 6시 수거)
- 수거량 많을 경우 오전 6시 이후까지 지연 가능
- 일요일/공휴일: 수거 가능 지역이면 수거 진행
- 비가 와도 정상 수거 진행
- **특정 시간 지정 불가**: 기사 동선이 매번 달라 시간 예약/보장 불가
- 주간 수거 불가 (야간 전용)

### 수거 신청 절차
1. 앱 > 수거 신청 > 날짜 선택 > 봉투 개수 지정 > 신청 완료
2. 신청일로부터 최대 2일 이내 일정만 선택 가능
3. 오후 10시 이후 기사 방문
4. 무게 측정 및 촬영
5. 수거 완료 시 앱 내 푸시 알림
6. 수거일 익일 오후 3시 자동결제

### 배출 필수 조건
- **봉투에 담아 배출 필수** (봉투 없이 품목만 배출 불가)
- **봉투 입구를 반드시 묶어서 배출** (테이프로 감싸는 것은 불가, 묶이지 않으면 테이프로 붙여서 배출)
- **손잡이 있는 봉투** 필수
- 신청 수량 = 배출 수량 **정확히 일치** 필수 (불일치 시 수거 진행 불가)
- 오후 10시 전까지 문 앞에 배출 필수
- 미배출 시(기사 방문 시 물품 없음): 기본요금 2,500원 청구

### 무게 제한
| 봉투 종류 | 최대 무게 | 초과 시 |
|-----------|-----------|---------|
| 일반 봉투 (80L) | **15kg 미만** | 수거 거부 |
| 대형 봉투 (150L/220L) | **25kg 미만** | 수거 거부 |

- 무게 초과 시 추가 패널티 없음 (수거 자체를 거부)
- 초과 시 여러 봉투로 분산 배출 후 재신청

### 무게 측정 방법
- 기사가 현장에서 전자 손저울(고리 걸이식)로 측정
- 가정용 저울과 약간의 오차 발생 가능 (오차 인정)
- 봉투당 1kg 미만 오차 허용
- 무게 측정 단위: 100g 단위로 반올림 계산
- 기사가 측정 사진 촬영 후 앱에 업로드
- 고객 측 무게 확인 방법: (봉투 들고 측정한 체중) - (기본 체중) = 봉투 무게
- 한 손으로 들었을 때 무거우면 분산 권장

### 배출 수량 제한 (주문당 수거 상한 — 2026-04-21 변경)
- **봉투(일반+대형 합계)**: 1주문당 최대 **10개까지 수거** (초과분 미수거)
- **대형 봉투**: 1주문당 **1개까지만 수거**. 2개 이상 배출되어도 1개만 수거하고 나머지는 미수거.
- **종이박스**: 1주문당 최대 **10개까지 수거** (초과분 미수거)
  - 종이박스 개수 산정: 큰 종이박스 안에 펼쳐진 작은 종이박스 여러 개가 들어있는 경우 **겉박스 포함 전체를 1개로 간주**
- 기존 "10개 초과 대량 배출 시 CS 승인 후 한번에 수거" 정책은 폐지. 상한 초과분은 수거되지 않음.

### 배출 위치
- **기본**: 문 앞 배출 (가장 안전, 권장)
- 문 앞이 아닌 위치 배출 시 필수 조건:
  1. 요청사항에 배출 위치 상세 기재
  2. 배출 위치 사진 업로드
  3. 필요시 심야 전화/세대 호출 동의
- 조건 미충족 시 미배출 처리, 기본요금 부과
- 외부 배출 시 분실/환경미화원 선수거 위험 있음 (고객 책임)
- 여러 봉투 분산 배출 불가 (한곳에 모아 배출 필수)
- 엘리베이터 없는 건물: **5층까지** 수거 가능, 6층 이상 수거 불가

### 수거 가능 품목
- **기본 원칙**: 화재 위험/의약품/산업 폐기물 제외한 대부분의 생활폐기물 수거 가능
- 혼합 배출 가능 (음식물, 재활용, 일반쓰레기 한 봉투에 혼합 OK)
- 봉투 내용물 품목 기재 불필요
- **확인된 수거 가능 품목**:
  - 의류, 신발, 가방, 이불, 베개, 침구류, 인형
  - 책, 종이류, 장난감, 레고
  - 주방용품: 후라이팬, 냄비, 압력밥솥, 그릇, 쟁반
  - 음식물 쓰레기 (밀봉 필수), 냉동식품, 김치, 견과류, 홍삼
  - 소형 가전: 전자레인지, 에어프라이어, 청소기, 제습기, 가습기, 헤어드라이기, 고데기, 다리미, 전기장판, 온수매트, 커피포트
  - 전자제품: 보조배터리, 충전기, 전선, 키보드, 마우스, 디지털카메라, 노트북, 멀티탭, 프린터
  - 건전지, 전구류, 형광등 (신문지 감싸기)
  - 화장품 (유통기한 지난 것 포함), 샴푸, 바디로션, 세제
  - 영양제, 건강기능식품, 비타민, 유산균 (의약품 아님)
  - 화분 (흙+식물 포함), 자갈
  - 캐리어, 골프백, 골프채
  - 스티로폼, 뽁뽁이
  - 유리병, 유리컵, 도자기 (신문지 감싸기)
  - 부탄가스, 스프레이캔 (거의 다 사용한 상태)
  - 라이터 (20개 이상 시 작은 박스에 담기)
  - 성인용품, 향수
  - 고양이 모래(벤토나이트), 고양이/강아지 배설물 (밀봉)
  - 택배 송장 등 개인정보 포함 물품 (안전 파기 처리)
  - 타일, 나무 판자, 합판 (무게 조건 내)
  - 카시트, 유모차, 유아용품
  - 텐트, 캠핑용품
  - 체중계, 블라인드
  - 스노우보드 데크, 롤매트, 요가매트
  - 셀프 바닥 타일
  - 달걀 (깨지지 않게 밀봉)

### 수거 불가 품목
- **의약품**: 병원 처방약, 한약, 약국 감기약, 연고, 인공눈물, 의료용 바늘류
- **화재 위험 제품**: 페인트 스프레이 (지정 폐기물), 라이터 연료
- **산업 폐기물**: 시멘트, 퍼티, 페인트, 액상실리콘 등 인테리어/건축 제품
- **화학약품류**: 자동차 세정제, 왁스류

### 특수 품목 배출 방법
| 품목 | 배출 방법 |
|------|-----------|
| 유리/칼/깨진 거울 등 날카로운 물품 | 신문지/작은 박스/쇼핑백으로 1차 밀봉 후 봉투에 삽입 |
| 음식물/액체류 (물 흐름 가능) | 비닐봉투로 1차 밀봉 후 봉투에 담기 |
| 냉동식품 (물맺힘 가능) | 비닐봉투 이중 밀봉 + 요청사항에 "냉동으로 인한 단순 물맺힘, 수거 진행 가능" 명시 |
| 박스/판지 | 내용물 제거 후 완전히 펼쳐서 배출 (접은 상태 불가) |
| PP박스/단프라 | 내용물 제거 + 전부 펼친 상태 + 노끈으로 묶기 + 수거 전 주문번호와 함께 채팅 전달 |
| 기저귀 | 밀봉 권장 |
| 라이터 20개 이상 | 작은 박스에 담아 봉투 안 배치 |

### 수거 거부 사유 (정책 미준수)
1. 무게 초과 (일반 15kg / 대형 25kg 초과)
2. 봉투 입구 미묶음 (내용물 새어나갈 우려)
3. 봉투 개수 불일치 (신청 수량 != 배출 수량)
4. 물 흐름 발생 (봉투 외부로 물이 흐르거나 새어나옴)
5. 커버링 봉투 아닌 봉투에 "커버링 수거" 메모 미부착
6. 손잡이 없는 봉투 (마대자루 등)
7. 너무 얇아 찢어질 위험 있는 봉투
8. 봉투 밖으로 3분의 1 이상 튀어나온 물품
9. 배출되지 않음 (기사 방문 시 봉투 없음)

## 3. 봉투 정책 (크기, 배송, 구매, 커버링봉투 vs 일반봉투)

### 봉투 종류 및 가격
| 봉투 종류 | 용량 | 무게 제한 | 가격 | 비고 |
|-----------|------|-----------|------|------|
| 일반 커버링 봉투 | 80L | 15kg 미만 | 300원/장 | 배송비 2,500원 별도 |
| 대형 커버링 봉투 | 150L (일부 220L 표기) | 25kg 미만 | 1,000원/장 | 배송비 2,500원 별도 |

- 150L 봉투 크기 참고: 신장 155~160cm 기준, 사무용 의자 정도
- 대형 봉투는 현재 **베타 테스트 기간** -- 무료 지급 중 (한시적)
- 대형 봉투 **1장씩만 신청 가능** (초기 운영 단계, 추후 개선 예정)
- 80L과 150L 동시 구매 불가 -- 각각 별도 신청 필요

### 신규 회원 혜택
- **무료 제공**: 80L 봉투 + 150L 봉투 + 배송비 무료 (1회 한정, 장수는 A/B 테스트 중이므로 구체적 장수 언급 금지)
- **조건**: 탈퇴 및 수거 이력이 없는 신규 회원만 해당
- 과거 탈퇴 후 재가입 시 무료 봉투/신규 쿠폰 미발급

### 수거 후 무료 봉투
- 수거 신청 시 "수거 후 무료봉투 안 주셔도 돼요" 체크 해제 시 기본 봉투 1장 무료 제공 (기사가 수거 시 함께 배송)
- 체크 시 봉투 미지급

### 봉투 배송
- **배송 기간**: 영업일 기준 3~5일 소요 (일반/대형 동일)
- **배송 방식**: 문고리에 걸거나 문 앞 바닥에 배치 / 택배업체(우체국, 두발히어로) 배송
- 대형 봉투는 일반 봉투와 별도 배송 (다른 날 도착 가능), 알림 미전송
- 배송 시작 후 취소 불가능
- 정확한 도착 날짜 안내 불가
- 재고 부족 시 추가 지연 가능

### 봉투 분실/누락 처리
- 무료 봉투 분실: **무료 재배송 불가** (유료 구매만 가능)
- 유료 구매 봉투 누락: 배송 업체 확인 후 재배송 (영업일 1~3일)
- 대체 방법: 일반 봉투 + "커버링 수거" 메모 부착으로 수거 가능

### 봉투 신청 경로
- 일반 봉투: 앱 > 봉투받기 (신규) / 봉투구매 (기존)
- 대형 봉투: 앱 > 봉투 구매 > 대형 폐기물 봉투(150L) 선택
- 봉투 구매와 수거 신청은 **별도 절차** (봉투 배송받은 후 별도로 수거 신청 필요)

### 커버링 봉투 없을 시 대체 사용 조건
- 일반 봉투(종량제/재활용 봉투 등) 사용 가능
- **필수 조건**:
  1. "**커버링 수거**" 메모 직접 작성하여 부착
  2. 손잡이 있는 봉투
  3. 15kg 미만
  4. 너무 얇지 않을 것 (찢어질 위험 없음)
  5. 물 흐름 없을 것
- 투명/반투명/검정 봉투 모두 가능
- **대형 폐기물 수거 시**: 반드시 커버링 대형 봉투만 사용 가능 (타사 봉투 불가)

### 봉투 배송 오류 처리
| 오류 유형 | 조치 |
|-----------|------|
| 봉투 오배송 (다른 주소) | 재발송 + 할인 쿠폰 |
| 봉투 개수 누락 | 초과 결제 취소 + 할인 쿠폰 + 재구매 권장 |
| 봉투 미배송 | 재배송 진행 |
| 봉투 사이즈 오류 (150L 신청 > 80L 수령) | 확인 후 재발송 (영업일 3~5일) |
| 불량 봉투 | 담당부서 보고 후 재확인/재발송 |

## 4. 가격/결제/쿠폰

### 요금 구조
- **기본요금**: 2,500원/건 (배송비 개념, 1회 수거 신청당 1회 부과, 봉투 수량 무관)
- **무게요금**: 140원/100g (선별 비용) = 1kg당 1,400원
- **총 요금** = 기본요금 2,500원 + (실제 무게 x 140원/100g)
- 무게 측정 후 청구 (실제 무게 기준)
- 박스 무게도 측정 대상 (무료 아님)

### 요금 예시
| 무게 | 계산 | 총액 |
|------|------|------|
| 5kg | 2,500 + 7,000 | 9,500원 |
| 10kg | 2,500 + 14,000 | 16,500원 |
| 15kg | 2,500 + 21,000 | 23,500원 |
| 25kg (대형) | 2,500 + 35,000 | 37,500원 |

### 할인
- **8kg 이상 배출**: 기본요금 20% 할인 (-500원)
- **구독 회원**: 매 주문건마다 5% 자동 할인

### 결제 시점 및 방법
- **자동결제**: 수거일 익일 오후 3시 등록된 카드로 자동 결제
- **수동결제**: 앱 > 이용내역 > 해당 수거건 상세페이지에서 직접 결제 가능 (오후 3시 전)
- **결제 수단**: 카드 결제, 카카오페이 간편결제만 지원
- **계좌이체**: 불가
- **카드 등록**: 앱 > 내정보 > 결제수단 > 새로운카드 등록 (카드번호, 유효기간, 생년월일 6자리, 비밀번호 앞 2자리 필요)
- 최소 1개의 카드 필수 등록 (시스템 요구사항)

### 카카오페이 결제 구조
1. 통장 > 카카오페이머니 충전 (1차)
2. 충전된 머니 > 커버링 결제 (2차)
- "결제 대기 상태"는 실제 결제가 아님 (혼동 주의)

### 환불 처리
- 카드 환불: 카드사 정책에 따라 **영업일 기준 1~5일** 소요
- 구독 환불: 영업일 기준 3~5일 소요

### 쿠폰 정책

#### 신규 가입 쿠폰
- 탈퇴 및 수거 이력 없는 신규회원만 지급
- 기본요금 무료 쿠폰 (무게요금은 별도 청구)
- 과거 탈퇴 후 재가입 시 미발급
- 만료 후 재발급 불가

#### 기본요금 무료 쿠폰 8장
- 신규 고객에게 기본요금 무료 쿠폰 8장 지급 (무게요금은 별도 청구)

#### 친구 초대 쿠폰
- 가입 후 24시간 내 초대 시 양쪽 모두 30% 할인 쿠폰 발급

#### 목요일 자동 쿠폰
- 매주 목요일 7시 이후 5% 할인 쿠폰 자동 발급 (늦어도 8시까지)

#### 미사용 28일 후 대청소 쿠폰
- 28일 미사용 시 30% 할인 쿠폰 발급

#### 100% 쿠폰
- 최대 5만원까지 무료 (초과분은 결제)

#### 쿠폰 적용 규칙
- **반드시 수거 신청 시 직접 선택/적용해야 함** (자동 적용 안 됨)
- 수거 완료 후 적용 불가
- 결제 후 적용 불가
- 1주문당 1개 쿠폰만 적용 가능
- 쿠폰 유효기간 내 수거 신청 시 적용 (실제 수거/청구 시기와 무관)
- 만료 쿠폰: 재발급 어려움 (단, 1회 재발급 가능한 경우 있음)
- 주문 취소 시 쿠폰 자동 복구되어야 하나, 시스템 오류로 수동 복구 필요한 경우 있음

#### 쿠폰 발급 사유
- 서비스 오류/불편: 기본요금 쿠폰 또는 할인 쿠폰
- 수거 지연, 출입 실패 (기사 오류), 무게 측정 오류, 미수거 (기사 실패), 주소 착오
- 지역 미지원 > 지역 오픈 시 신규 쿠폰 재발급

### 결제 오류 처리

#### 카드 등록 실패 ("비밀번호 오류 횟수 초과")
1. 카드 비밀번호 변경 후 재등록 시도
2. 계속 오류 시 카드사 고객센터 연락 (제한 해제 요청)

#### 무게 입력 오류
- 기사가 무게 잘못 입력한 경우: 실제 무게 기준으로 수정
- 과다 청구 시 부분 취소 + 사과 쿠폰 지급

### 영수증
- PG사 변경으로 앱 내 미제공 -- 카드사 홈페이지/앱/고객센터에서 직접 확인 발급
- 사업자: 세금계산서 별도 발급 (이메일 등록 필요)

## 8. 출입/접근 문제 처리

### 공동현관 출입 정보
- 비밀번호/호출 정보를 요청사항에 **정확히** 기재 필수
- 비밀번호만 적기 (X): "4561"
- 정확한 기재 (O): "비밀번호 버튼 누르고, 4561" / "# 버튼 누르고 16040808 입력"
- 종 모양 버튼/열쇠 버튼/호출 버튼 등 **모든 입력 정보 명시** 필수
- 출입 정보 수정: 시스템에서 직접 수정 불가 > 취소 후 재신청 필수

### 심야 세대호출
- 신청 시 "심야 호출 동의" + 호출 방법 상세 기재
- 동의 시 출입 문제 발생 시 기사가 고객에게 전화
- 고객이 전화 받지 않으면 미수거 처리
- 미동의 시 기본 설정 = "동의 없음"

### QR코드 출입 건물
- 수거 신청 시 QR코드 이미지 1장 첨부 가능 (상세페이지 > 수거 품목 이미지란)

### 카드키 건물
- 기사가 카드키 작동 불가 > 추가 출입 정보 필수

### 출입 실패 시 처리
- 기사 착오 시: 취소 처리 + 재신청 가능 + 할인 쿠폰 발급, 기본요금 미청구
- 정보 오류 시: 기사가 경비실 호출 또는 다른 방법 시도, 실패 시 미수거
- 고객 정보 정확했으나 출입 실패 시: 기본요금 면제 + 쿠폰 지급

### 층수 제한
- 엘리베이터 없는 건물: **1~5층까지** 수거 가능
- 6층 이상 (옥상 포함): 수거 불가
- 엘리베이터 있는 건물: 제한 없음

| 봉투 분실 (무료 봉투) | 수량 취소 (재발송 불가) |
| 쿠폰 만료 | 1회 재발급 (상황별 판단) |
| 배차 조회 시스템 지연 | 20% 쿠폰 발급 |

### 분실물 처리
- 수거 후 선별장 도착 시 분실물 찾기 어려움
- 하루 수천 건 이상 처리
- 중요물품은 실수로 넣지 말 것

## 10. 앱/시스템 관련

### 수거 신청 수정/취소
- **수정 기능**: 최근 앱 업데이트로 수정 버튼 제한적 제공 중
- **대체 방법**: 주문 취소 > 재신청 (유일한 방법)
- **취소 가능 시간**: 밤 10시 이전까지 (10시 이후 배차 완료 시 취소 불가)
- 배차 전: CS 승인 필수로 수량 수정 가능
- 배차 후: 현장 기사 판단에 따라 일부 조정 가능

### 주문 상태 흐름
- 접수신청 > 수거예정 (오후 10시 이후) > 수거완료
- 방문 중에는 상태 변경 없음

### 앱 네비게이션
- 수거 신청: 앱 > 수거 신청
- 봉투 신청: 앱 > 봉투구매 > 대형 폐기물 봉투 (150L)
- 이용내역 확인: 앱 > 이용내역 > 지난 수거 탭
- 서비스 지역 확인: 앱 > 내정보 > 서비스 지역
- 주소 변경: 앱 > 좌측 상단 주소명 클릭 > 주소 변경 > "이 주소를 기본 주소로 설정" 선택
- 쿠폰 확인: 앱 > 내정보 > 쿠폰
- 결제수단: 앱 > 내정보 > 결제수단
- 탈퇴: 앱 > 내정보 > 내정보 설정 > 탈퇴하기
- 1:1 문의: 앱 > 내정보 > 1:1 문의하기

### 계정 관리

#### 주소 변경
- 앱 좌측 상단에서 직접 수정 가능
- 새 주소 추가 후 **반드시 '대표 주소'로 설정** 필수
- 미설정 시 이전 주소로 계속 수거됨

#### 탈퇴
- 앱 > 내정보 > 내정보 설정 > 탈퇴하기
- **조건**: 진행 중인 주문건이 모두 "완료" 상태여야만 탈퇴 가능
- 탈퇴 시 모든 정보 삭제 (결제 정보, 주소, 개인정보, 카드 정보 완전 삭제)
- 자동결제 중단

#### 번호 변경/계정 이전
- 필수 정보: 기존 이용 번호, 기존 주소 및 출입 방법, 신규 이용 번호
- 신규 번호 가입 시 기존 계정 탈퇴 + 로그아웃 필요
- 데이터 이전 작업에 시간 소요

### 보안 주소 등록
- 카카오 API 미지원 주소: 근처 주소로 등록 후 상세정보에 정확한 주소 기재

### 개인정보
- 상담 중 수집된 휴대폰번호: 최대 3일 보유 후 자동 삭제
- 전자기기 초기화: 고객이 직접 진행 (커버링에서 개인정보 파기 불가)

### 시스템 오류 사례
- 대형 봉투 자동 재발송 오류 (2024년): 기존 신청 고객 대상 오류 배송, 비용 없음
- 오발송 알림톡 (수거 실패, 결제 미완료 등 테스트 발송 오류): 무시해도 됨

### 회원 등급
- **산정 기준**: 최근 71일(약 10주) 동안의 결제 완료된 수거 주문 수
- **확정일**: 매주 월요일 오후 4시

## 11. 대형폐기물 (방문수거 -- 별도 서비스)

### 서비스 개요
- 봉투에 들어가지 않는 대형 물품 전용 (매트리스, 침대, 소파, 장롱, 책상, 냉장고, 피아노 등)
- 커버링 앱의 일반/대형 봉투 수거와는 **완전히 별도 서비스**
- 초기 운영 단계, 안정적 제공 위해 단계적 준비 중

### 비용
- 지자체 스티커보다 높은 가격대
- 출장비 포함하여 견적 제공

### 신청 방법
- 앱 내 "방문수거" 견적 요청 창구 이용
- 또는 support@covering.app 이메일
- 담당 부서에서 순차적으로 연락
- 사진 등록 필요: 웹사이트 > 신청 조회 > 주문 수정에서 사진 등록
- 사진 확인 후 일정 확정
- 예약 확정 메시지 없으면 예약 안 된 상태

### 대형 봉투 수거 vs 방문수거 구분
- 150L/220L 봉투에 담기는 크기: 대형 봉투 수거 (앱 신청)
- 봉투에 담기지 않는 크기: 방문수거 서비스 (별도 견적)

## 13. 기타

### 오피스(사업장) 서비스
- **최소 수거 조건**:
  - 주 1회 수거: 키트 2개 이상 필수
  - 주 2회 수거: 키트 1개 이상 필수
- **요금 계산**: 키트 단가 x 키트 개수 x 수거 회수 = 월 정산 비용
- 정기 수거만 (비주기적 불가)

### 오피스 봉투 가격 (VAT 포함)
- 80L: 9,350원
- 120L: 13,200원

### 파티룸/상업용 공간
- 정기 수거 미지원
- 일회성 수거만 가능 (앱 서비스 지역 내)

### 대량 수거 정책 (2026-04-21 변경)
- **주문당 수거 상한 고정**: 봉투 10개(일반+대형 합계, 대형은 그 중 1개) + 종이박스 10개
- 상한 초과분은 **일반/대형 봉투 수거로 처리되지 않음**.
- 종이박스 개수 산정: 큰 종이박스 안에 펼쳐진 작은 종이박스 여러 개가 담겨 있어도 **전체를 1개로 간주**
- **10개 초과 대량 배출은 커버링 방문수거 서비스로 안내**: 카카오톡 방문수거 채널(https://pf.kakao.com/_bxgWhX/chat)에서 견적 요청
- 기존 "10개 초과 시 CS 승인 후 한번에 수거" 방식은 폐지.

### 50% 할인 광고
- 첫 이용 시 50% 할인 광고 있음 (적용 조건 불명확)

### 상담 중 욕설/비속어
- 포함 시 상담 중단 가능

### 수거 신청 시 사진 첨부
- 시스템상 사진은 1개만 첨부 가능
- 추가 사진은 채팅으로 전달

### 봉투 구분 규칙 (커버링 봉투 자체)
- 150L 봉투: 커버링 프린팅 없음, 25kg 제한
- 80L 봉투: 커버링 프린팅 있음, 15kg 제한
- 대형 봉투 사용 후 "일반 봉투 수거"로 신청하면 일반 정책(15kg) 적용됨
- 대형 폐기물 수거 신청 시에만 25kg 기준 적용

### 대형 봉투 혼합 배출
- 최신 정책상 대형 품목 + 일반 쓰레기 함께 담기 가능 (25kg 미만)
- 단, 일부 데이터에서 "잡동사니 금지" 언급 (대형 폐기물 전용 용도 권장)

### 수거 지연 공지
- 공식 공지사항 미제공 (요청 시만 개별 안내)
- 수거 지연 시 오전 6시에 지연 알림톡 발송 (사전 안내 불가)

### 개인정보 처리
- 택배 송장, 개인정보 포함 물품: 안전 파기 처리 (외부 유출 방지 후 파쇄/폐기)
- 서류 파쇄: 별도 안내 없이 일반 수거로 처리

### 보안 관련
- 공동현관 비밀번호가 우편물에 노출되는 경우: 배송업체 개선 요청 필요

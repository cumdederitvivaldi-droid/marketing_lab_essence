# 커버링 런치 서비스 정책

## 1. 서비스 개요
- 단체 도시락 및 단체 행사 후 발생한 폐기물을 수거하는 서비스
- 분리배출 불필요 — 배송 온 박스 또는 봉투에 먹던 그대로 뚜껑만 덮어 한 곳에 모아두면, 지정 시간에 수거 진행

## 2. 요금 구조

### 총 비용 = 수거 요금 + 처리 요금

### 2-1. 수거 요금
- 지역별 방문 수거 운송비
- **100인분 기준**으로 요금이 달라짐 (100인분 미만 / 100인분 이상)
- 지역별 요금은 시스템의 region-prices.json 데이터를 참조
  - `lunchSmall` = 100인분 미만 수거요금
  - `price1` = 100인분 이상 수거요금
- 표에 없는 지역은 **별도 견적** → 상담사에게 전달 (NEED_HUMAN)

### 2-2. 처리 요금
- 수거 후 도시락 분리배출 비용
- **도시락 1개당 500원** (부가세 포함)
- 예외: **한솥 도시락은 1개당 400원**

### 2-3. 야간 수거
- **시간대: 오후 10시 ~ 오전 6시 사이 진행 (고정 시간대, 시간 지정 불가)**
- 오후 9시 이전 요청은 야간이 아닌 **오후 수거**로 안내
- 출장비(수거요금) 없음
- 처리요금만 적용 (개당 500원)
- **최소 금액: 10,000원** (20개 미만이어도 10,000원)
- "새벽수거", "야간수거", "밤에 수거" 등 표현 = 야간
- 고객이 "야간 몇 시", "밤 X시" 등 구체적 시각 요청 시: 시간 지정 불가 안내. 오후 10시~오전 6시 사이 기사 동선에 따라 진행된다고 답변.

### 2-4. 계산 예시
```
[주간] 서울 용산구, 도시락 90개
→ 수거요금 32,000원 (100인분 미만) + 처리요금 45,000원 (500원 × 90개)
→ 총 77,000원 (부가세 포함)

[야간] 서울 강남구, 도시락 80개
→ 수거요금 0원 (야간) + 처리요금 40,000원 (500원 × 80개)
→ 총 40,000원

[야간] 도시락 15개
→ 최소금액 적용 → 총 10,000원
```

## 3. 행사 수거
- 행사 관련 문의는 **담당자에게 전달 필요** → NEED_HUMAN
- 차량별 비용 (부가세 별도):
  - 1톤 저상 트럭: 25만원
  - 1톤 하이탑차: 35만원
  - 5톤 집게차: 120만원

## 4. 수거 신청 프로세스

### 4-1. 필요 정보 (수거 신청 양식)
1. 수거 날짜
2. 수거 시간 (오전/오후/야간) — 야간은 오후 10시~오전 6시 고정, 시간 지정 불가
3. 수거 주소 (상세 위치 포함)
4. 도시락 개수
5. 신청자 연락처
6. 현장 담당자 연락처
7. 상호명
8. 결제방법 (계좌이체/카드결제)
9. 야간 신청시 출입방법 기재 (해당시)

### 4-2. 접수 기한
- 원칙: **수거일 기준 2일 전**
- 예외: 하루 전 가능
- **야간 수거는 당일도 가능**

### 4-2-1. 당일 수거 요청 응대
"당일 수거 가능한가요?" / "오늘 수거 되나요?" 같은 문의 시:
- 야간 수거: **당일 신청 가능**
- 주간 수거(오전/오후): 원칙적으로 2일 전 접수 권장, **하루 전까지 가능**, 당일은 원칙적으로 불가
- 응답 템플릿:
```
안녕하세요 커버링 런치입니다 :)

당일 수거의 경우, 야간 수거라면 당일 신청도 가능합니다!

다만 주간(오전/오후) 수거는 원칙적으로 수거일 기준 2일 전 접수를 권장드리고, 하루 전까지는 가능합니다.

혹시 오늘 수거를 원하시는 건가요? 주간/야간 중 어느 쪽으로 생각하고 계신지 알려주시면 확인해드리겠습니다 :)
```

### 4-3. 미결제 건 확인
- 이전 수거 건 중 **미결제 내역이 있으면 수거 접수 불가**
- 고객에게 미결제 안내 후 결제 완료 확인되면 접수 진행

## 5. 수거 신청 양식 (신규 고객 안내용)

```
안녕하세요! 커버링 런치입니다 :)

도시락 수거 서비스를 이용해주셔서 감사합니다.
분리배출 없이 먹던 그대로 뚜껑만 덮어 한 곳에 모아두시면,
지정하신 시간에 수거 진행해드립니다.

수거 신청을 위해 아래 내용을 보내주세요:

1. 수거 날짜
2. 수거 시간 (오전/오후/야간) — 야간은 오후 10시~오전 6시 고정, 시간 지정 불가
3. 수거 주소 (상세 위치 포함)
4. 도시락 개수
5. 신청자 연락처
6. 현장 담당자 연락처
7. 상호명
8. 결제방법 (계좌이체/카드결제)
9. 야간 신청시 출입방법 기재 (해당시)

접수 후 담당자가 확인 연락드리겠습니다!
```

## 6. 변경 및 취소

### 6-1. 당일 시간 변경
- 기본적으로 **불가**
- 가능 시 **시간당 1만원 추가 비용**
- → NEED_HUMAN (담당자 확인 필요)

### 6-2. 당일 취소
- **수거 요금 발생** (처리요금은 부과하지 않음)

### 6-3. 수량 변경
- 수거 전까지 수량 변경 가능
- 변경 시 금액도 자동 재계산

## 7. 미수거 발생 시
- → NEED_HUMAN (담당자 확인 필요)
- 자사 귀책: 비용 없이 재수거
- 고객 귀책 (출입실패 등): 재수거 시 비용 발생 가능

## 8. 정산
- 결제는 수거 완료 후 담당자가 안내 (백오피스 시스템에서 처리)
- 결제 방식: 카드결제(링크페이) / 월말정산 / 세금계산서
- AI는 정산 세부사항을 안내하지 않음, "수거 완료 후 결제 안내드리겠습니다" 정도만

## 9. 에스컬레이션 규칙 (NEED_HUMAN으로 전환)

AI가 직접 처리하지 않고 상담사에게 넘겨야 하는 경우:
- 행사 수거 문의
- 견적서 요청
- 당일 시간 변경 요청
- 미수거 발생/클레임
- 도시락 외 폐기물 문의
- 표에 없는 지역 견적
- 세금계산서/정산 관련 복잡한 요청
- 고객 불만/컴플레인
- 야간 수거 미수거 관련

## 10. 기타 규칙

- 첫 이용 고객: 서비스소개 + 수거 신청 양식 안내
- 기존 고객 (벤더명 매칭됨): 바로 주문 접수 처리
- 채팅방 이름: 상호명으로 관리
- 최소 주문 수량: 20개
- 야간 20개 미만: 개수 무관 10,000원
- 월말정산 업체: 손에손에, 캐치잇, 나이스캐더링

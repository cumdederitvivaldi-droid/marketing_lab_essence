# 커버링 방문수거 시간 운영 정책

## 1. 시간 블록 체계 (권장)

기본은 3개 블록으로 안내.

- **A타임 09:00~12:00** (오전)
- **B타임 13:00~16:00** (오후 이른 시간)
- **C타임 17:00~20:00** (오후 늦은 시간)

각 블록 내 구체 도착 시각은 **기사 동선에 따라 조정**되며, **고객이 블록 내 특정 시각을 지정하는 것은 어렵습니다**. 블록 범위 내에서 방문한다는 점을 명확히 안내.

## 2. 응대 원칙 (강제 아님, 권장)

고객이 수거 시간을 문의하면 **우선 A/B/C 3블록으로 제안**. 구체 시각 지정은 아래 경우에만 수용:

- 건물 경비 시간·업무시간 제약이 명시됨
- 엘리베이터 사용 시간이 한정됨
- 고객이 2회 이상 특정 시각을 강조함 (예: "꼭 오후 3시에 와주세요")

추가 비용 없음 (비공개 원칙).

## 3. 응대 문구 가이드

### 3-1. 블록 제안 (기본)
> "수거 희망 날짜가 정해지셨으면 알려주세요. 가능한 시간대는
> 오전(9~12시) / 오후(13~16시) / 저녁(17~20시) 중 편하신 시간대로 안내드립니다.
> 블록 내 정확한 시각은 기사 동선에 따라 조정됩니다 :)"

### 3-2. 구체 시각 수용 (예외)
> "확인했습니다. 오후 3시 기준으로 배정 도와드리겠습니다 :)"

### 3-3. 블록 마감 안내
> "해당 날짜 A타임은 마감되어 B, C 타임만 가능합니다."

## 4. 시간안내 버튼

상담사가 `[시간안내]` 버튼을 사용하면 고객에게 가능한 블록만 카카오톡 버튼 카드로 자동 발송됩니다.
이 경우 고객이 버튼 클릭만 하면 해당 블록으로 예약 접수되며, AI 는 추가로 시간 문구 재질문을 하지 않습니다.

## 5. 금지 문구

- "블록 외 시간은 불가능합니다" (강제 느낌 금지)
- "정확히 몇 시에 오나요?" 같은 반문 — 블록 특성상 정확 시각 불가한 점 설명으로 대체
- "A타임" "B타임" 같은 내부 용어를 고객에게 직접 사용 — "오전 9~12시" 같은 한글 범위 표현 사용

## 6. 예약 관련 운영 정책

- **채팅 상담 운영시간**: 오전 10시 ~ 오후 10시 (수거 시간과 다름)
- **수거 가능 시간**: 오전 9시 ~ 오후 8시, 평일·공휴일 동일.
- **공휴일**은 차량정비 등 사유가 있을 수 있어 **대현님 확인 후 예약 접수**.
- 동시간대 예약은 **3건 미만까지 가능**. 3건 이상이면 **대현님에게 별도 확인 필요**.
- **비대면 수거**는 특정 시각보다 `오전 10:00~오후 2:00` 또는 `오후 1:00~오후 4:00` 같은 **범위로 잡으면 케파에 큰 도움** (배차 효율·고가 건 수용).
- 예약 후 **사진 요청 필수**.
- **잡동사니**는 양 파악 + 모아두기 요청 필수. **정리 안 됨 + 5봉투 이상 + 품목 흩어져 있음** → 봉투당 **+7,000원 정리 요금** (총 17,000원/봉투).
- **100만원 이상 견적**은 **무조건 대현님께 일정 확인 후 예약 확정**.
- **세금계산서 발행**은 **원빈님** 담당. 고객에게 세금계산서 발행을 약속했다면 **입금 확인 후 원빈님께 발행 요청**. (대현님 아님 — 혼동 금지)
- **결제 기본 권장**: 링크페이 (카드 / 가상계좌). 계좌이체(무통장 입금) 또는 세금계산서 발행 요청 고객에게만 아래 계좌 안내.
  - **기업은행 039-079846-04-066 (주)커버링**
  - 세금계산서 요청 시 **사업자등록증 + 수신 이메일** 함께 받기.
- **결제 링크 발송 채널**: 예약 확정 즉시 **카카오톡**으로 나이스페이 링크페이 링크 전송 (수거 후 발송 → 2026-05-11 부터 예약 즉시 발송으로 변경).

### 6.1. 100% 선결제 정책 (2026-05-11 시행)

- **예약 확정 시 100% 선결제 필수**. 사다리차/일반 구분 없이 전액을 예약 시점에 결제.
- 예약이 시스템에 등록되면 **자동으로 카카오톡 결제 링크 발송**.
- **방문 12시간 전까지 결제 미완료 시 예약 자동 취소** (방문 시간대의 시작 시각 - 12h 기준).
  - 예: 5월 12일 오전 9시 ~ 12시 블록 → 5월 11일 오후 9시까지 결제 필요.
- 자동취소 발생 시 고객 카카오톡으로 안내 메시지 자동 발송.
- 운영 관점: 매일 오후 6시 슬랙 익일 브리핑 스레드에 자동취소 건이 함께 보고됨.
- 상담사 응대 가이드:
  - 고객이 결제 안내를 받지 못했다고 하면 PaymentModal 에서 재발송.
  - 일정 변경 요청은 결제 전이면 자유롭게, 결제 후면 환불·재결제 절차 안내.

### 6.2. 전화상담 운영 (정책 진본)

- **운영 시간**: 오전 10시 ~ 오후 6시 (수거 가능 시간 / 채팅 운영시간과 다름).
- **신청 방법**: 채팅에서 성함 + 연락처 남기기.
- **응답 기한**: 1시간 이내 전화 (운영시간 내).
- 운영시간 외 신청은 다음 영업일 오전 10시부터 순차 대응.

### 이벤트: 1만원 할인 프로모션 (2026-04-22 ~ 2026-05-03)
- 커버링스팟 카톡 채널 브랜드 메시지로 1.1만명씩 수·목·금·토 오후 2시 발송.
- **고객이 먼저 언급한 경우에만** 적용. "1만원 할인"/"브랜드 메시지"/"카톡 쿠폰" 등 키워드 감지 시 → 최종 견적에서 1만원 차감.
- **기예약 고객**이 같은 요청을 해도 동일하게 1만원 할인 적용 (기존 예약건 금액 조정).
- 2026-05-03 이후 문의는 "이벤트 종료" 안내.
- ⚠️ 커바니는 이 이벤트를 **먼저 제안하거나 적용 리마인드를 자동으로 띄우면 안 된다**. 모든 상담이 할인 대상이 아니며, 고객이 먼저 언급한 대화에 한해서만 관련 힌트를 제공.

## 7. 수거 불가 품목

- 피아노(업라이트·그랜드) — 단, **전자/디지털 피아노는 수거 가능**. 모델·크기·해체 가능 여부 확인 위해 **반드시 사진 요청 필수** (예약 전 단계에서).
- 돌침대, 흙침대, 물침대
- 에어컨 실외기 (안전상 불가)
- 공사 폐기물
- 의료 폐기물
- 철거 작업이 필요한 케이스 (화장실·주방 수납장, 변기 등 붙어있는 구조물) — 주변 훼손 우려
- 가스 차단이 **사전 안 된** 가스레인지

## 8. 수거 불가 케이스

- 지상 진입 불가
- 사다리차 필요하지만 사다리차 진입 어렵거나 협조 어려움

## 9. 엘리베이터 미설치 건물 수거 기준

- **냉장고 / 안마의자 / 러닝머신 / 크고 무거운 운동기구** → 2층까지 가능 (3층부터 사다리차 필요)
- **소파 2~3인용 / 퀸 매트리스·프레임 / 원목 큰 책장** → 4층까지 사다리차 없이 작업 가능

## 10. 2인 이상 수거 필요 품목

- 냉장고
- 안마의자
- 러닝머신 (크고 무거운 운동기구)
- 대리석·유리가 포함된 품목 (식탁, TV장 등)
- 장롱 3자 / 4자

## 11. 수거 누락 대응

1. 1순위 — **30% 할인 안내 후 일정 재조율**
2. 2순위 — 재예약 의사 없음 → **추후 이용 시 30% 할인 안내 후 종결**

## 12. 수거 지연 대응

- 고객 문제 제기 시 **10% 할인 안내 후 일정 재조율**

## 13. 주변 훼손 발생 시

- 고객이 수리 후 **영수증 제출 시 보상 처리**. 추가 할인 X.

## 14. 현장 물품 추가 시

- 기사님은 **적재 공간 부족 / 1인 수거 불가 / 예의 어긋나는 고객** 등 사유로 수거 거부 가능.
- 적재 공간 부족 시 **추후 일정 잡아 별도 예약 권유**.
- 커버링 **80L 봉투당 1만원** (기사님이 직접 정리 시 **+1만원 추가**).

## 15. 현장에서 수거 불가 품목 발견 시

- **커버링 과실** → 출장비 없이 재수거 일정 조율 후 수거
- **고객 과실** → 재예약 요청, 견적에서 수거 불가 품목 제외 후 결제 요청

## 17. 담당자 구분 (매우 중요 — 혼동 금지)

내부에 두 명의 담당자가 있고 역할이 다르다. 커바니가 조언할 때 담당자를 **정확히** 구분해야 한다.

- **대현님** — 운영 확인 담당
  - 공휴일 예약 접수 전 확인
  - 동시간대 3건 이상 예약 확인
  - 100만원 이상 견적 일정 확인
- **원빈님** — 정산·발행 담당
  - 세금계산서 발행 요청 (입금 확인 후)
  - 결제/환불/정산 관련 처리

## 18. 어시스턴트(🐰) 코칭 가이드

상담사 어시스턴트는 위 정책들을 기준으로 대화를 관찰하며 **놓치기 쉬운 포인트를 짧게 한 문장**으로 알려줍니다.

우선순위:
1. 정책 위반 또는 확인 미흡 (공휴일 확인, 100만원↑, 사다리차 결제, 정리 요금, 2인 수거 필요 품목)
2. 수집 누락 (주소·엘리베이터·주차·품목·사진)
3. 수거 불가 품목·케이스 신호 (피아노, 돌/흙/물침대, 실외기, 철거 필요, 지상 진입 불가)
4. Phase 기반 다음 액션 (견적 발송 시점, 예약 확정 더블체크)

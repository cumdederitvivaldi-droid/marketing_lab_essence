covering-invite-v2 환경변수 정리
================================

배포 시 서버에서 직접 세팅 필요.
- 위치: /shared/apps/covering-invite-v2/.env 또는 /shared/.env
- 로컬 개발: .env.local (gitignore 대상)


1. NEXT_PUBLIC_KAKAO_JAVASCRIPT_KEY
   - 용도: 카카오톡 공유 SDK 초기화 (초대자 "친구에게 알려주기" 버튼)
   - 필수: 카카오 공유 사용 시 필수
   - 없으면: 클립보드 복사로 폴백
   - 담당: 환희 — 카카오 개발자 콘솔에서 공용 키 발급 후 세팅
   - 참고: 인준님(개발팀장)한테 키 전달 필요

2. AIRBRIDGE_TOKEN
   - 용도: 피초대자 가입 시 Airbridge API로 트래킹 링크 생성 (누가 누구를 초대했는지 추적)
   - 필수: 원래 설계 의도상 필요
   - 없으면: 앱스토어 링크로 대체 — 가입은 되지만 트래킹 정확도 떨어짐
   - 담당: 기존 Vercel 프로젝트(covering-invite) Settings > Environment Variables에서 가져오기


3. FLARELANE_PROJECT_ID
   - 용도: FlareLane 프로젝트 식별 (서버 라우트에서 푸시 이벤트 전송)
   - 필수: V2 코드 참조 — 누락 시 푸시 트래킹 누락
   - 값: /shared/.env 공통 변수 재사용 (covering-labs 환경변수 레지스트리)

4. FLARELANE_API_KEY
   - 용도: FlareLane API 인증 (서버사이드)
   - 필수: FLARELANE_PROJECT_ID와 함께 필수
   - 값: /shared/.env 공통 변수 재사용

5. GCP_PROJECT
   - 용도: GCP 프로젝트 식별 (`covering-app-ccd23`) — BigQuery/GCS 경로
   - 필수: V2 서버 라우트에서 참조
   - 값: /shared/.env 공통 변수 재사용 (VM 서비스 계정 자동 인증)

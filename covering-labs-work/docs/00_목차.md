## 이 서버는 뭔가요?

각자 다른 환경(로컬 맥미니, 무료 클라우드 등)에서 실험을 진행하다 보니 성능/안정성 이슈가 있어서, 회사 공용 GCP 서버를 만들었습니다.

**용도:**
- 24시간 돌아가야 하는 배치 스크립트
- Next.js / NestJS 서버 자동 배포 (GitHub push → 자동 빌드/실행)
- 로컬 맥미니 서버 대체
- POC / 실험 환경

**`@covering.app` 구성원이면 누구나 SSH 접속 가능합니다.**

## 빠른 시작

```bash

gcloud init

# 2. SSH 접속
gcloud compute ssh --zone "asia-northeast3-a" \
  "covering-labs-instance-20260306-050059" \
  --project "covering-app-ccd23"
```

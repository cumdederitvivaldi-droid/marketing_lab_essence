# 링퀴즈 80cm 미만 방문수거 제외 수정 플랜

> 유형: 플랜
> 작성일: 2026-05-24
> 상태: 완료

## 목표

링퀴즈에서 길이 80cm 미만 물품이 무게 때문에 방문수거로 추천되지 않게 한다.

## 현황 분석

현재 `weight-over-25` 규칙이 길이 기반 봉투 규칙보다 먼저 적용되어 `UNDER_80 + OVER_25` 조합이 방문수거로 추천된다.

## 구현 계획

### 단계별 작업

- [x] `UNDER_80 + 25kg 이상/혼자 들기 어려움` 조합을 일반 봉투 여러 장 추천으로 우선 처리한다.
- [x] 가전·가구/혼합 카테고리 방문수거 테스트는 80cm 초과 조건으로 명확히 수정한다.
- [x] 정책 매트릭스에 80cm 미만 무거운 혼합 품목 방문수거 제외 사례를 추가한다.
- [x] 테스트, 타입체크, 빌드를 실행한다.

## 완료 기준

- 72cm, 25kg 이상, 복수 카테고리 선택 시 방문수거가 아니라 일반 커버링 봉투 여러 장이 추천된다.
- 기존 80cm 초과·150cm 초과 방문수거 정책은 유지된다.
- 관련 테스트와 빌드가 통과한다.

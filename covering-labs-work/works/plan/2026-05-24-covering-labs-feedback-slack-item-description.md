# 링퀴즈 피드백 Slack 물품명 추가 플랜

> 유형: 플랜
> 작성일: 2026-05-24
> 상태: 완료

## 목표

링퀴즈 결과 페이지에서 피드백이 제출될 때 Slack 알림에 사용자가 수기로 입력한 물품명을 함께 표시한다.

## 현황 분석

현재 피드백 payload는 수기 입력 여부와 글자 수만 전달하고, Slack 메시지도 원문을 표시하지 않는다.

## 구현 계획

### 단계별 작업

- [x] 피드백 state snapshot에 제한된 길이의 수기 물품명을 포함한다.
- [x] Slack 메시지에는 수기 물품명을 표시하고, Supabase 저장 row에서는 원문을 제외한다.
- [x] 기존 피드백 저장 및 Slack 발송 테스트를 갱신한다.

## 완료 기준

- Slack 메시지 본문에 수기 입력 물품명이 포함된다.
- Supabase에 저장되는 `app_state`에는 수기 입력 원문이 남지 않는다.
- 관련 테스트와 타입체크가 통과한다.

## 1. 서비스 개요

### 서비스 정의
- **커버링**: 생활폐기물 야간수거 서비스
- 고객이 앱으로 수거 신청 → 야간(22:00~06:00) 기사 방문 → 봉투 수거 → 무게 측정 → 후 결제
- **선 수거, 후 결제** 방식

### 서비스 지역
- 수도권 + 천안 + 세종 + 청주 + 대전 지역 운영
- 서울: 매일 수거 가능
- 인천 서구: 신현원창동 제외 매일 수거
- 경기 고양시 덕양구: 효자동, 고양동, 관산동 서비스 불가
- 세종: 조치원읍, 한솔동, 도담동, 아름동, 종촌동, 고운동, 보람동, 새롬동, 대평동, 소담동, 다정동, 해밀동, 반곡동, 나성동, 어진동 등 (화/금/일)
- 청주시: 상당구(월/수/금/일), 서원구(월/수/금/일), 흥덕구(화/금/일), 청원구(월/목/토) — 오송읍 포함
- 대전: 서구(화/목/토/일), 유성구(월/수/금/일), 동구(월/수/목/토), 중구(화/목/토/일), 대덕구(월/수/목/토) — 전 구 서비스
- 미지원: 대구, 부산, 경남 밀양, 경기 양평, 충남 아산 음봉면 등
- 지역별 수거 가능 요일 상이 (앱 > 내정보 > 서비스 지역 확인)
- 미지원 지역 회원: 탈퇴 없이 계정 유지 시, 지역 오픈 때 신규 쿠폰 재발급

### 상담 채널
- **유선 상담 불가** — 채팅 상담만 가능 (채널톡)
- 상담 운영시간: 오전 10시 ~ 오후 10시 30분
- 종료된 채팅은 누락 위험 → 새 채팅으로 재문의 권장

## 2. 수거 규칙

### 수거 시간
- **수거 시간대**: 오후 10시(22:00) ~ 익일 오전 6시(06:00)
- **당일 수거 마감**: 오후 10시 이전 신청 필수
- 오후 10시 이후 신청 → 익일 수거
- 수거량 많을 경우 오전 6시 이후 지연 가능
- 일요일/공휴일: 수거 가능 지역이면 정상 진행
- 비 와도 정상 수거
- **특정 시간 지정 불가** (기사 동선 매번 상이)
- 주간 수거 불가 (야간 전용)

### 수거 신청 절차
1. 앱 > 수거 신청 > 날짜 선택 > 봉투 개수 지정 > 신청 완료
2. 신청일로부터 최대 2일 이내 일정만 선택 가능
3. 오후 10시 이후 기사 방문
4. 무게 측정 및 촬영
5. 수거 완료 시 앱 푸시 알림
6. **수거일 익일 오후 3시 자동결제**

### 배출 필수 조건
- **봉투에 담아 배출 필수** (봉투 없이 품목만 배출 불가)
- **봉투 입구 반드시 묶기** (테이프만으로 감싸기 불가, 묶이지 않으면 테이프 보조 가능)
- **손잡이 있는 봉투** 필수
- 신청 수량 = 배출 수량 일치 필수. **불일치 시 수량에 맞게 일부만 수거** (초과 봉투는 수거하지 않음)
- 오후 10시 전까지 문 앞 배출 필수
- 미배출 시(기사 방문 시 물품 없음): **기본요금 2,500원 청구**

### 무게 제한
| 봉투 종류 | 최대 무게 | 초과 시 |
|-----------|-----------|---------|
| 일반 봉투 (80L) | **15kg 미만** | 수거 거부 |
| 대형 봉투 (220L) | **25kg 미만** | 수거 거부 |

- 초과 시 추가 패널티 없음 (수거 자체를 거부)
- 여러 봉투로 분산 배출 후 재신청

### 무게 측정
- 기사가 전자 손저울(고리 걸이식)로 현장 측정
- 가정용 저울과 약간의 오차 발생 가능
- 봉투당 1kg 미만 오차 허용
- 100g 단위 **올림** 계산 (예: 1,480g → 1,500g으로 청구)
- 측정 사진 촬영 후 앱 업로드

### 배출 수량 제한 (주문당 수거 상한 — 2026-04-21 변경)
- **봉투(일반+대형 합계)**: 1주문당 최대 **10개까지 수거** (초과분은 수거 안 함)
- **대형 봉투**: 1주문당 **1개까지만 수거**. 2개 이상 배출된 경우에도 1개까지만 수거하고 나머지는 미수거.
- **종이박스**: 1주문당 최대 **10개까지 수거** (초과분은 수거 안 함)
  - ⭐ **종이박스 개수 산정 규칙**: 큰 종이박스 안에 펼쳐진 작은 종이박스 여러 개가 담겨 있는 경우 → **전체를 1개로 간주** (겉박스 1개 = 1개)
- **대형 봉투 + 일반 봉투 동시 신청 시**: 합계 10개 상한 유지 (대형 1 + 일반 9 등)
- **대형 폐기물**: 수거 신청 1건당 대형 봉투 1개만 수거 가능. 나머지 품목은 일반 봉투로 별도 신청 필요.
- ⚠️ 기존 "10개 초과 배출 시 CS 승인 후 한번에 수거" 정책은 폐지됨. 주문당 상한을 초과하는 배출은 수거되지 않음.

### 배출 위치
- **기본**: 문 앞 배출 (권장)
- 문 앞 외 위치 시 필수 조건:
  1. 요청사항에 배출 위치 상세 기재
  2. 배출 위치 사진 업로드
  3. 필요시 심야 전화/세대 호출 동의
- 미충족 시 미배출 처리, 기본요금 부과
- 외부 배출 시 분실/환경미화원 선수거 위험 (고객 책임)
- 여러 봉투 분산 배출 불가 (한곳에 모아 배출)
- 엘리베이터 없는 건물: **5층까지** 수거 가능, 6층 이상 불가

### 수거 가능 품목
- **원칙**: 화재 위험/의약품/산업 폐기물 제외한 대부분의 생활폐기물
- 혼합 배출 가능 (음식물+재활용+일반쓰레기 한 봉투에 OK)
- 봉투 내용물 품목 기재 불필요
- **수거 가능**: 의류, 신발, 가방, 이불, 베개, 침구류, 인형, 책, 종이류, 장난감, 주방용품(후라이팬, 냄비, 압력밥솥), 음식물(밀봉 필수), 소형 가전(전자레인지, 에어프라이어, 청소기, 제습기, 가습기), 전자제품(보조배터리, 충전기, 전선, 키보드, 마우스, 노트북), 건전지, 전구류, 형광등(신문지 감싸기), 화장품, 샴푸, 세제, 영양제/건강기능식품(의약품 아님), 화분(흙+식물 포함), 캐리어, 골프백/골프채, 스티로폼, 유리병/유리컵/도자기(신문지 감싸기), 부탄가스/스프레이캔(거의 사용 완료 상태), 라이터(20개 이상 시 박스 담기), 고양이모래, 동물 배설물(밀봉), 택배 송장 등 개인정보 포함 물품(안전 파기), 타일, 나무판자, 카시트, 유모차, 텐트/캠핑용품, 체중계, 블라인드, 스노우보드 데크, 요가매트

### 수거 불가 품목
- **의약품**: 병원 처방약, 한약, 약국 감기약, 연고, 인공눈물, 의료용 바늘류
- **화재 위험**: 페인트 스프레이(지정 폐기물), 라이터 연료
- **산업 폐기물**: 시멘트, 퍼티, 페인트, 액상실리콘 등 인테리어/건축 제품
- **화학약품류**: 자동차 세정제, 왁스류

### 특수 품목 배출 방법
| 품목 | 배출 방법 |
|------|-----------|
| 유리/칼/깨진 거울 등 날카로운 물품 | 신문지/박스/쇼핑백 1차 밀봉 후 봉투에 삽입 |
| 음식물/액체류 | 비닐봉투 1차 밀봉 후 봉투에 담기 |
| 냉동식품 (물맺힘) | 비닐봉투 이중 밀봉 + 요청사항에 "냉동 단순 물맺힘, 수거 진행 가능" 명시 |
| 박스/판지 | 내용물 제거 후 완전히 펼쳐서 배출 |
| PP박스/단프라 | 내용물 제거 + 전부 펼친 상태 + 노끈 묶기 + 주문번호와 채팅 전달 |
| 기저귀 | 밀봉 권장 |
| 라이터 20개 이상 | 작은 박스에 담아 봉투 안 배치 |

### 수거 거부 사유
1. 무게 초과 (일반 15kg / 대형 25kg)
2. 봉투 입구 미묶음
3. 봉투 개수 불일치
4. 물 흐름 발생 (봉투 외부로 물 새어나옴)
5. 커버링 봉투 아닌 봉투에 "커버링 수거" 메모 미부착
6. 손잡이 없는 봉투 (마대자루 등)
7. 너무 얇아 찢어질 위험 있는 봉투
8. 봉투 밖으로 1/3 이상 튀어나온 물품
9. 배출되지 않음 (미배출)

## 3. 봉투 정책

### 봉투 종류 및 가격
| 봉투 종류 | 용량 | 무게 제한 | 가격 | 비고 |
|-----------|------|-----------|------|------|
| 일반 커버링 봉투 | 80L | 15kg 미만 | **300원/장** | 배송비 2,500원 별도 |
| 대형 커버링 봉투 | 220L | 25kg 미만 | **현재 무료** (한시적) | 배송비 2,500원 별도 |

- 220L 크기 참고: 신장 155~160cm 기준, 사무용 의자 정도
- 대형 봉투 **베타 테스트 기간** — 무료 지급 중
- 대형 봉투 **1장씩만 신청 가능**
- 80L과 220L 동시 구매 불가 (각각 별도 신청)

### 앱 수거 신청 봉투 선택지
- ⚠️ 앱에서 수거 신청 시 선택할 수 있는 봉투는 **2가지뿐**:
  1. **"쓰레기 봉투"** = 일반 커버링 봉투(80L) 또는 "커버링" 메모 붙인 일반 쓰레기봉투 **모두 가능** (앱에서 이 둘을 구분하지 않음)
  2. **"대형 커버링 봉투"** = 220L 대형 봉투
- ⚠️ "일반봉투로 접수" vs "커버링봉투로 접수"라는 구분은 **존재하지 않음**. 고객이 이런 구분을 언급하면 혼동이 있는 것이므로, "쓰레기 봉투" 접수는 커버링 봉투든 '커버링' 메모 붙인 일반 봉투든 모두 사용 가능하다고 안내할 것.

### 봉투 규격
- 80L 봉투: 커버링 프린팅 있음, 15kg 제한
- 220L 봉투: 커버링 프린팅 있음, 25kg 제한
- ⚠️ **대형 봉투(220L)를 사용하고 "쓰레기 봉투"로 신청하면 수거 불가**. 반드시 "대형 커버링 봉투"로 신청해야 함.

### 신규 회원 혜택
- **무료 제공**: 80L 봉투 + 220L 봉투 + 배송비 무료 (1회, 장수는 A/B 테스트 중이므로 구체적 장수 언급 금지)
- **조건**: 탈퇴 및 수거 이력 없는 신규 회원만
- 과거 탈퇴 후 재가입 시 무료 봉투/신규 쿠폰 미발급

### 수거 후 무료 봉투
- 수거 신청 시 "수거 후 무료봉투 안 주셔도 돼요" 체크 해제 시 기본 봉투 1장 무료 제공
- 체크 시 봉투 미지급

### 봉투 배송
- **배송 기간**: 영업일 기준 **1~3일** (일반/대형 동일)
- **배송비**: **2,500원** (신규 회원 첫 배송 무료)
- 배송 방식: 문고리/문 앞 배치 또는 택배(우체국, 두발히어로)
- 대형 봉투는 일반과 별도 배송 (다른 날 도착 가능), 알림 미전송
- 배송 시작 후 취소 불가
- 정확한 도착 날짜 안내 불가
- 재고 부족 시 추가 지연 가능

### 봉투 신청 경로
- 일반 봉투: 앱 > 봉투받기 (신규) / 봉투구매 (기존)
- 대형 봉투: 앱 > 봉투 구매 > 대형 폐기물 봉투(220L) 선택 (현재 **설문 폼 접수** 방식)
  - ⚠️ 대형 봉투는 별도 **신청 완료 알림톡이 발송되지 않음**. "응답이 성공적으로 제출되었습니다" 화면이 뜨면 정상 신청된 것으로 안내.
- **봉투 구매와 수거 신청은 별도 절차** (봉투 배송 후 별도로 수거 신청)

### 커버링 봉투 없을 시 대체 사용 조건
- 일반 봉투(종량제/재활용 봉투 등) 사용 가능
- **필수 조건**:
  1. "**커버링 수거**" 메모 직접 작성하여 부착
  2. 손잡이 있는 봉투
  3. 15kg 미만
  4. 찢어질 위험 없을 것
  5. 물 흐름 없을 것
- 투명/반투명/검정 봉투 모두 가능
- **대형 폐기물 수거 시**: 커버링 대형 봉투만 사용 가능 (타사 봉투 불가)

### 봉투 분실/누락 처리
| 상황 | 처리 |
|------|------|
| 무료 봉투 분실 | 재배송 불가, 유료 구매만 가능 |
| 유료 봉투 누락 | 배송 업체 확인 후 재배송 (영업일 1~3일) |
| 봉투 오배송 | 재발송 + 할인 쿠폰 |
| 봉투 개수 누락 | 초과 결제 취소 + 할인 쿠폰 + 재구매 권장 |
| 봉투 사이즈 오류 | 확인 후 재발송 (영업일 1~3일) |
| 불량 봉투 | 담당부서 보고 후 재확인/재발송 |

## 4. 가격/결제

### 요금 구조
- **기본요금**: **2,500원**/건 (1회 수거 신청당 1회만 부과, 봉투 수량과 무관)
  - ⚠️ **봉투를 여러 개 배출해도 기본요금은 2,500원 단 1회만 부과됨** (봉투당 X, 건당 O)
  - 예: 봉투 3개 배출 → 기본요금 2,500원 × 1회 = 2,500원 (7,500원 아님)
- **무게요금**: **140원/100g** (= 1kg당 1,400원) — 모든 봉투의 총 무게 합산, **100g 단위 올림** 적용
  - 예: 실측 350g → 400g(4단위)으로 올림 → 140원 × 4 = 560원
- **총 요금** = 기본요금 2,500원 + (올림 적용된 무게 x 140원/100g)
- 박스 무게도 측정 대상

### 요금 계산 공식 (중요)
총 결제 = 기본요금 2,500원 + (ceil(무게g ÷ 100) × 140원)
- ceil = 100g 단위 올림. 4,300g → 43단위, 350g → 4단위, 5,050g → 51단위

### 요금 예시
| 무게 | 무게단위(÷100) | 무게요금 | 기본요금 | 총액 |
|------|------|------|------|------|
| 4,300g | 43 | 43 × 140 = 6,020원 | 2,500원 | 8,520원 |
| 5kg | 50 | 50 × 140 = 7,000원 | 2,500원 | 9,500원 |
| 10kg | 100 | 100 × 140 = 14,000원 | 2,500원 | 16,500원 |
| 15kg | 150 | 150 × 140 = 21,000원 | 2,500원 | 23,500원 |
| 25kg (대형) | 250 | 250 × 140 = 35,000원 | 2,500원 | 37,500원 |

기본요금 무료 쿠폰 적용 시: 기본요금 0원 → 무게요금만 결제 (예: 4,300g → 6,020원)

### 할인
- **8kg 이상 배출**: 기본요금 20% 할인 (-500원)
- **구독 회원**: 매 주문건 5% 자동 할인

### 결제
- **자동결제**: 수거일 익일 오후 3시 등록된 카드로 자동결제
- **수동결제**: 앱 > 이용내역 > 지난수거 > 해당 건 수거 상세페이지에서 결제 가능 (오후 3시 자동결제 전 직접 결제)
- **결제 수단**: 카드 결제, 카카오페이만 지원 (계좌이체 불가)
- **카드 등록**: 앱 > 내정보 > 결제수단 > 새 카드 등록 (카드번호, 유효기간, 생년월일 6자리, 비밀번호 앞 2자리)
- 최소 1개 카드 필수 등록

### 카카오페이
1. 통장 → 카카오페이머니 충전 (1차)
2. 충전된 머니 → 커버링 결제 (2차)
- "결제 대기 상태"는 실제 결제 아님 (혼동 주의)

### 환불
- 카드 환불: **영업일 1~5일** (카드사별 상이)
- 구독 환불: **영업일 3~5일**
- 오인수거 환불: **1~3영업일**

### 영수증
- PG사 변경으로 앱 내 미제공 — 카드사 홈페이지/앱에서 직접 발급
- 사업자: 세금계산서 별도 발급 (이메일 등록 필요)

### 결제 오류
- 카드 등록 실패 ("비밀번호 오류 횟수 초과"): 카드 비밀번호 변경 후 재등록 / 카드사 고객센터 제한 해제
- 무게 입력 오류: 실제 무게 기준 수정 + 부분 취소 + 사과 쿠폰

## 5. 쿠폰 정책

### 신규 가입 쿠폰

- 가입 후 24시간 내 초대 시 양쪽 모두 30% 할인 쿠폰

### 목요일 자동 쿠폰
- 매주 목요일 7시 이후 5% 할인 쿠폰 자동 발급 (**기존 회원만**)
- 신규 회원: 3개월 동안 **기본요금 무료 쿠폰**으로 대체 (5% 쿠폰 대신)

### 미사용 28일 후 대청소 쿠폰
- 28일 미사용 시 30% 할인 쿠폰

### 100% 쿠폰
- 최대 5만원까지 무료 (초과분 결제)

### 쿠폰 적용 규칙
- **유효기간**: 쿠폰별 상이 (전체가 1주일은 아님 — 각 쿠폰의 유효기간 개별 확인 필요)
- **반드시 수거 신청 시 직접 선택/적용** (자동 적용 안 됨)
- **수거 완료 후, 결제 전**: 수거 상세페이지 하단에서 쿠폰 적용 가능
- **결제 완료 후**: 쿠폰 적용 불가
- 1주문당 1개 쿠폰만
- 쿠폰 유효기간 내 수거 신청 시 적용 (실제 수거/청구 시기 무관)
- 만료 쿠폰 재발급 어려움 (1회 재발급 가능한 경우 있음)
- 주문 취소 시 쿠폰 자동 복구 (시스템 오류로 수동 복구 필요한 경우 있음)
- 쿠폰 확인: 앱 > 내정보 > 쿠폰

### 쿠폰 발급 사유
- 서비스 오류/불편: 기본요금 쿠폰 또는 할인 쿠폰

## 9. 출입/접근 문제

### 공동현관 출입 정보
- 비밀번호/호출 정보를 요청사항에 **정확히** 기재
- 잘못된 예: "4561" → 올바른 예: "비밀번호 버튼 누르고, 4561"
- 모든 입력 정보(종/열쇠/호출 버튼 등) 명시 필수
- 출입 정보 수정: 직접 수정 불가 → **취소 후 재신청**

### 심야 세대호출
- 신청 시 "심야 호출 동의" + 호출 방법 상세 기재
- 동의 시 출입 문제 발생 시 기사가 고객에게 전화
- 미응답 시 미수거 처리
- 기본 설정: "동의 없음"

### 특수 출입
- QR코드: 수거 신청 시 QR 이미지 1장 첨부 가능 (수거 품목 이미지란)
- 카드키: 기사 사용 불가 → 추가 출입 정보 필수

### 출입 실패 처리
- 기사 착오: 취소 + 재신청 + 할인 쿠폰, 기본요금 미청구
- 정보 오류: 기사가 경비실 호출 등 시도, 실패 시 미수거
- 고객 정보 정확했으나 출입 실패: 기본요금 면제 + 쿠폰

### 층수 제한
- 엘리베이터 없는 건물: **1~5층** 수거 가능, 6층 이상(옥상 포함) 불가
- 엘리베이터 있는 건물: 제한 없음

## 12. 대형폐기물 (방문수거 -- 별도 서비스)

### 개요
- 봉투에 들어가지 않는 대형 물품 전용 (매트리스, 침대, 소파, 장롱, 냉장고, 피아노 등)
- 커버링 앱의 일반/대형 봉투 수거와 **완전히 별도 서비스**

### 비용
- 출장비 포함 견적

### 신청 방법
- 앱 > "방문수거" 견적 요청 또는 카카오톡 방문수거 문의(https://pf.kakao.com/_bxgWhX/chat)

### 구분 기준
- 220L 봉투에 담기는 크기 → 대형 봉투 수거 (앱)
- 봉투에 담기지 않는 크기 → 방문수거 서비스 (별도 견적)

## 13. 기타

### 오피스(사업장) 서비스
- 최소 수거: 주 1회 = 키트 2개 이상 / 주 2회 = 키트 1개 이상
- 요금: 키트 단가 x 키트 개수 x 수거 회수 = 월 정산
- 정기 수거만 (비주기적 불가)
- 봉투 가격: 80L 9,350원, 120L 13,200원 (VAT 포함)

### 파티룸/상업용 공간
- **정기 수거 지원** (커버링 오피스 서비스 이용)

### 대량 수거 정책 (2026-04-21 변경)
- **주문당 수거 상한 고정**: 봉투 10개(일반+대형 합계, 대형은 그 중 최대 1개) + 종이박스 10개
- 상한을 초과하는 배출물은 **일반/대형 봉투 수거로는 처리되지 않음**.
- 종이박스 개수 산정: 큰 박스 안에 펼쳐진 작은 박스 여러 개가 들어있어도 **전체를 1개로 간주**
- **10개 초과 대량 배출은 커버링 방문수거 서비스로 안내**: 카카오톡 방문수거 채널(https://pf.kakao.com/_bxgWhX/chat)에서 견적 요청
- ⚠️ 기존 "10개 초과 시 CS 승인 후 한번에 수거" 방식은 폐지.

### 수거 지연 안내
- 공식 공지 미제공 (개별 안내만)
- **1차 알림 (오전 6시)**: 수거 미완료 상태 시 지연 알림톡 발송 (알림만)
- **2차 알림 (오전 8시)**: 여전히 미완료 시 할인 쿠폰 **자동 발급** + 지연 알림톡 발송
- 2시간 이내 별도 안내 없으면 낮 시간 중 마무리 예정

### 수거 신청 시 사진
- 시스템상 1개만 첨부 가능, 추가 사진은 채팅 전달

### 상담 중 욕설/비속어
- 포함 시 상담 중단 가능

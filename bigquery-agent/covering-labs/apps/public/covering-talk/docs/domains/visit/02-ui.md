### 커바니 (Kovani) — 방문수거 전용 어시스턴트

- 위치: 우측 하단 마스코트, 드래그 가능
- 트리거: 고객(`role:user`) 새 메시지 1.5초 debounce 후 자동 호출
- 표시: 1줄 코칭 + 📖 정책 버튼
- API: `/api/conversations/[sessionId]/assistant-hint` → Haiku → `{hint, section}`
- 정책 모달: AI 가 `section` 반환 → 📖 클릭 시 PolicyModal 이 해당 heading 으로 자동 스크롤 + 노란 플래시
- 정책 원문: `lib/ai/pickup-policy.md` (프롬프트·모달 공동 소스)
- UX 영속: 위치·숨김 모두 `localStorage` (`assistantBuddyPos`, `assistantBuddyDismissed`)
- z-index: 500 (일반 모달 9999 < 커바니 < PolicyModal 10000)

## 공유 컴포넌트

- `components/PaymentModal.tsx` — 방문수거 결제 모달 (별도 도메인 접근 사용 안 함). 결제 이력 entry 마다 "📋 링크" 복사 버튼 (결제완료 entry 제외). 안내 메시지 템플릿에 latest reqId 가 있으면 NicePay URL 라인 자동 삽입.
- `components/conversations/LadderPrepaymentModal.tsx` — 사다리차 선결제 모달. 금액 입력 + "기존 견적에 포함" 토글 (ON 시 부모 totalPrice 차감 미리보기) + 카카오톡 발송 + 발송 후 URL 복사. CustomerPanel 의 `needLadder` 인 예약 카드 하단 🪜 버튼에서 띄움.
- `components/AuditLogPanel.tsx` — 감사 로그 패널 (모든 도메인 공통)
- `components/ui/*` — Shadcn UI 프리미티브

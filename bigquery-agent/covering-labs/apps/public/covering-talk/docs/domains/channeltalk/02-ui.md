## 메시지 표시 규칙

채널톡 message.options 필드에 따라 UI 분기:

| options 조합 | UI |
|---|---|
| `private` + `silentToUser` | 내부대화 (상담사 간 메모) — 노란색 표시 |
| `private` + `silentToUser` + `silentToManager` + `immutable` | 시스템 로그 — 필터링 |
| `actAsManager` | 매니저로 표시 (고객에게도 보임) |
| `silentToUser` + `doNotPost` | 워크플로우 / 봇 내부 처리 — 텍스트 있으면 표시 |

자세한 분기는 [`lib/channeltalk/types.ts`](../../../lib/channeltalk/types.ts).

## 파일 / 이미지 표시

- 파일: `key` 필드 → URL `https://cf.channel.io/{key}`
- 썸네일: `previewKey` → `https://cf.channel.io/{previewKey}`
- 이미지: `type === "image"` → 인라인 렌더링
- 그 외 (PDF·docx 등): 다운로드 링크
- 일부 파일은 CDN 서명 필요 → `/api/channeltalk/file` 프록시 사용

## SuggestPanel 동작

1. 채팅 선택 시 자동 호출 (디바운스)
2. 옵션:
   - 일반 호출 (`/api/channeltalk-ai/suggest`)
   - 스트리밍 (`/api/channeltalk-ai/suggest/stream`) — 파이프라인 단계별 실시간 표시
3. 결과:
   - 카테고리 라벨
   - 추천 답변 (Sonnet 생성 + Haiku 톤)
   - 매크로 후보 (있으면)
   - RAG 인용 정책 섹션
4. 상담사 액션:
   - **채택** → 입력창에 자동 채워짐
   - **편집** → 수정 후 발송
   - **무시** → 직접 작성

## ToolPanel 액션

- 자동 태깅 (`/api/channeltalk/chats/[chatId]/auto-tag`)
- 태그 추가/삭제 (`/api/channeltalk/chats/[chatId]/tags`)
- 차량 자동 추천 (`/api/channeltalk/chats/[chatId]/vehicle-auto`)
- 보류 (`/api/channeltalk/chats/[chatId]/snooze`)
- 종료 (`/api/channeltalk/chats/[chatId]/close`)
- 담당자 배정 (`/api/channeltalk/chats/[chatId]/assign`)
- 메시지 삭제 (`/api/channeltalk/chats/[chatId]/delete-message`) — Desk API 사용

## 분석 페이지 (`/channeltalk/analytics`)

- 카테고리별 카운트 (시간 범위 필터)
- 응답시간 분포 (채널톡 cases API)
- 상담사별 처리량
- AI 채택률 (`channeltalk_reply_logs` 기반)

## botName (메시지 발송 표시명)

- API 키는 채널 단위 → 모든 발송이 같은 매니저로 표시되는 문제
- `botName` 쿼리 파라미터로 표시명 override
- 현재: 로그인한 상담사의 `name` 을 botName 으로 사용 → 고객에게 상담사 이름 그대로 표시

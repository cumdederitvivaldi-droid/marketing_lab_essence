※ 쿠폰입력률 = DB 실전환 / MMP 전환 × 100

※ 광고+쿠폰 CAC = (광고비 + 쿠폰비) / MMP 전환 유저 수  [기존 보고서 방식]

## 4. 결론 요약
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

[광고 효율 (MMP 기준)]
 - DCJ50 CPA 61,959원 → 3종 중 최저, 전환량(56명)도 최다 → 광고 효율 최우수
 - DCJ25 CPA 157,080원 → 광고 자체 도달 약함 (설치 96건, 가장 적음)
 - DCJ100 CPA 196,069원 → CPA 최고이나 쿠폰 유인력 강해 전환률은 준수

[실전환 ROI (DB 기준)]
 - DCJ50: 순매출 0원 (쿠폰 비용 = 매출, 광고비 3.4M원 별도)
 - DCJ100: 순매출 -96,460원 (쿠폰 비용 > 매출, 광고비 1.8M원 별도)
 - 세 쿠폰 모두 첫 주문 단계에서는 적자 → 재구매 전환이 핵심
 - 완료율 95.8%로 서비스 이탈 이슈는 없음

[쿠폰 코드 실입력률 = 28.2%]
 - MMP 전환 78명 중 22명만 실제 DCJ 쿠폰 코드 입력
 - 나머지 56명은 광고 보고 설치 후 쿠폰 없이 구매했거나 다른 쿠폰 사용
 - 랜딩/온보딩에서 DCJ 코드 입력 유도 UX 개선 시 실전환율 제고 가능

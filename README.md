# marketing-lab

도형의 마케팅 실험실 — 비주얼 작업 자동화 플러그인 모음.

설치: `/plugin marketplace add hound600al/marketing-lab-26-05-09`

## 등록된 플러그인

| 플러그인 | 설명 |
|----------|------|
| [visual-tools](plugins/visual-tools/) | 무드보드 캡처(`/moodboard`) + 파일 정리(`/sort-files`) + Playwright MCP |
| [interview-deck](plugins/interview-deck/) | 면접 발표덱 병렬 제작 파이프라인(`/interview-deck`) + deck-researcher·deck-deployer 에이전트 |

## interview-deck 빠른 설치

```
/plugin marketplace add hound600al/marketing-lab-26-05-09
/plugin install interview-deck@marketing-lab-26-05-09
```
그 뒤 `"OO 회사 OO 직무 면접 발표덱 만들어줘"` 또는 `/interview-deck` 로 사용.
